# C Learning Repository

This repository documents my process of learning C with a focus on embedded systems programming.

Since I already understand programming fundamentals and C-like syntax (from languages such as C# and Java), I focused mainly on learning C-specific concepts

## Pointer Tutorial

  The folder `pointer_tutorial` contains code based on the pointer tutorial by Ted Jensen:
  https://github.com/jflaherty/ptrtut13/tree/master
  This material is used for educational purposes while I work through pointer concepts in depth.

## LibAlternatives

The `LibAlternatives` folder contains my own implementations of common C standard library functions, such as:

- `strlen()`
- `strstr()`
- `strrev()`
- `strchr()`
- and other similar utilities

The goal is to understand how these functions work internally by reimplementing them from scratch.
