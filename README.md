# C Learning Repository

This repository documents my process of learning C, with a focus on embedded systems programming.

Since I already understand programming fundamentals and syntax from languages such as C# and Java, I focused mainly on concepts that are specific to C, especially memory management, pointers, low-level data representation, and bit manipulation.

## Pointer Tutorial

The `pointer_tutorial` folder contains code based on Ted Jensen's pointer tutorial:

https://github.com/jflaherty/ptrtut13/tree/master

This material is used for educational purposes while I study pointers and memory concepts in depth.

## LibAlternatives

The `LibAlternatives` folder contains my own implementations of common C standard library functions, organized into a custom static library.

Implemented functions include:

- `strlen()`
- `strstr()`
- `strrev()`
- `strchr()`
- and other similar utilities

The goal is to understand how these functions work internally by recreating them from scratch and learning how C libraries are structured.

## Bit Operations

The `BitOperations` folder contains exercises focused on:

- bit manipulation
- bit masks
- flags
